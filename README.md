# Push-Based React Nx Workshop

## Installation Instructions

**System Requirements**

* `node > 18`
* `npm > 8`

**Clone and install**

```bash
git clone https://github.com/push-based/react-movies-app.git

cd react-movies-app
npm install

# (optional) if the step before didn't work, please try the following
npm install --force
```

**Run the application**

```bash
npm run start
```
